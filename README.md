# 🌳 Родовое дерево Шаповаловых

Интерактивный сайт для просмотра семейного дерева.

## Возможности

- **Все люди** — карточки с фото и поиском по имени
- **Дерево** — визуальное дерево для любой персоны (прародители → внуки)
- **Статистика** — поколения, фамилии, количество пар

## Как открыть локально

```bash
# Простой HTTP сервер (нужен для загрузки photos/ и persons.json)
python3 -m http.server 8080
# Открыть: http://localhost:8080
```

Или через VS Code Live Server / любой другой локальный сервер.

## Данные

- `persons.json` — 172 человека с типизированными связями
- `photos/` — 70 фотографий

## Сборка

Не требует сборки. Чистый HTML + DaisyUI CDN + Tailwind CDN.
